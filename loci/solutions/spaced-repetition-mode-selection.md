# Spaced Repetition: Free Recall > MCQ for Weak Topics

## Discovery (2026-02-16)

During GARP RAI quiz session, user failed an MCQ on fairness measures (M3-fairness-measures, 44% accuracy) despite saying "I can recall the decision tree." Immediately after, switched to free-recall format (scenario presented, user types answer without options) — scored 8/10 on the same topic.

## Why MCQ Fails for Weak Topics

MCQ provides answer options that enable **pattern matching** — the user recognises the right answer from context clues without actually retrieving the concept. This masks weak retrieval and gives false confidence.

Free recall forces **production** — the user must generate the answer from memory. Errors are honest and expose the actual gap.

## Threshold

- **< 50% accuracy:** Definition drill (present term, user types what they remember)
- **50-69% accuracy:** Scenario free-recall (present scenario, user types answer + reasoning, no options)
- **>= 70% accuracy:** MCQ (exam-like conditions, retrieval is solid enough to test under pressure)

## Also Discovered

Narrative/visual mnemonics ("four people at a bank") stick better than tabular decision trees for this user. Multiple overlapping frameworks (table + decision tree + signal words + quick reference) compete under exam pressure. Pick one primary anchor, demote the rest.
