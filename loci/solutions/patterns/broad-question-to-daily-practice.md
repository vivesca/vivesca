# Broad Question → Daily Practice Pattern

When a broad development question ("how can I grow?") needs to become actionable.

## The sequence that worked

1. **Start with honest frameworks** — not generic advice, grounded in the person's actual situation
2. **Iterate through pushback** — each "but what about..." deepens the framework (smart questions → genuine questions, selling → diagnosis, pushback → three types)
3. **Stress-test with councils** — but only after the framework exists. Councils on blank-slate questions produce generic advice. Councils on specific frameworks surface real blind spots (power > skills reframe)
4. **Filter council output with judgment** — not everything novel is useful. Rate each finding: gold / worth noting / premature / wrong timing
5. **Create time-bound action plans** — checkboxes with review dates, not open-ended principles
6. **Operationalise into daily practice** — a skill with active recall beats a note you read once. The GARP quiz pattern (prompt → respond → evaluate → log) transfers to soft skills via scenario practice
7. **Set an expiry** — daily practice stops when real reps begin. Don't let the training programme outlive its usefulness

## Key insight

The councils' biggest contribution wasn't the advice — it was reframing from "skills gap" to "power/mandate problem." This only worked because the skills framework was already built, giving them something specific to challenge. Blank-slate councils produce blank-slate advice.

## When to use

When someone (including yourself) asks a broad growth/development question and you need it to land as daily behaviour, not just insight.
