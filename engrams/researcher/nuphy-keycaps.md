## NuPhy Air75 v3 Keycap Compatibility

### Switch type
- Gateron LP 3.0 low-profile switches (hot-swappable)
- 3.5mm travel (unusually long for low-profile)
- MX-style stem (low-profile variant), NOT standard-height MX

### Keycap compatibility rules
- REQUIRES low-profile MX stem keycaps
- Standard-height MX keycaps (Cherry, OEM, SA, GMK etc.) do NOT fit correctly — wrong height/proportion, won't seat properly on LP switches
- Kailh Choc keycaps: incompatible (different stem entirely)
- All NuPhy Air series keycaps (from Air60, Air75, Air96, all versions) are cross-compatible with Air75 v3 — confirmed by NuPhy official Threads post

### Profiles NuPhy sells for Air series (LP MX)
- **nSA** — NuPhy's original low-profile profile; flat/uniform-ish, shorter height; ships as default on Air75 v3
- **Berry** — introduced 2024; sculpted/stepped rows (like Cherry profile but lower); taller than nSA; Cherry-inspired ergonomics
- **Shine-through (nSA)** — ABS, backlight-optimised, same profile as nSA

## Berry Profile — Key Facts

- Introduced by NuPhy in 2024
- Designer/origin: NuPhy proprietary (not a third party)
- Material: PBT dye-sublimated legends
- Stem: low-profile MX (Gateron LP compatible)
- Profile shape: sculpted/stepped rows — each row has different slope/angle, like Cherry profile but scaled down for LP heights
- Height: taller than nSA (no exact mm published in sources found)
- Description: "low-profile version of Cherry profile" — multiple sources use this framing
- Base kit price: $24 (WoB/BoW); full sets like Oblivion $49; Peach Blossom $49

## Berry on Air75 v3 — Compatibility

CONFIRMED COMPATIBLE. NuPhy explicitly lists Air75 v3 in the supported boards for all Berry profile sets (Oblivion, WoB/BoW, Peach Blossom, Beat Boy, Berry Trackday).

No known fitment issues reported.

## User Feedback — Berry vs nSA on Air75 v3

- Multiple users prefer Berry over stock nSA:
  - Better texture (PBT vs stock double-shot PBT — slight difference in surface feel)
  - Less prone to shine/grease pickup
  - Taller profile reduces accidental multi-key presses (nSA is very flat)
  - Better sound (fuller keystroke sound due to extra height)
  - Higher WPM reported by some users
- One user noted stock nSA was "a bit too low" and Berry resolved it
- No negative fitment or compatibility complaints found across sources

## Alternatives for Air75 v3 (LP MX stem)

Options are limited because LP MX keycaps are a small market:
1. NuPhy nSA profile sets (stock profile, many colorways)
2. NuPhy Berry profile sets (upgrade pick — recommended by community)
3. NuPhy nSA Shine-through (ABS, RGB-optimised)
4. KBDiy 9009 Doys (LP PBT, retro aesthetic) — mentioned in keeb-finder.com
5. Standard MX keycaps: NOT compatible due to height mismatch
6. Keychron LSA: NOT compatible (different stem ecosystem — Keychron LP boards)

## Misinformation Patterns
- "Standard MX keycaps fit low-profile keyboards" — FALSE; stem fits physically but proportions are wrong and caps will sit too high
- "Berry profile is the same height as nSA" — FALSE; Berry is noticeably taller (Cherry-inspired vs NuPhy's flat LP default)
- Exact height specs (mm per row) not publicly documented — don't invent numbers
